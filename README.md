# vrisa-backend

 Proyecto desarrollado para la gestión, visualización y participación ciudadana en estaciones de monitoreo ambiental, integrando un frontend en Next.js, Supabase como backend-as-a-service, y conexión con el repositorio backend en Node.js.

---

## Instalación de Dependencias
npm install
 o 
yarn install

## Ejecutar el entorno local
npm run dev

## Abrir el navegador en
[Localhost 3000](http://localhost:3000/dashboard/admin/users)


## Paquetes principales usados:

```bash
next: Framework React SSR
react y react-dom: Base del frontend
@supabase/supabase-js: Cliente para interactuar con Supabase
tailwindcss: Estilos rápidos y adaptables
typescript: Tipado estático y robustez
eslint y postcss: Calidad de código y compatibilidad
```

## Tecnologías principales

```bash
Next.js 14 (App Router)
TypeScript
Supabase
TailwindCSS
Node.js / Express (backend complementario)
```
